Чем отличается Tinkoff ID от Tinkoff ID для юрлиц?

Это один и тот же сервис, который упрощает вход на ваш сайт и в приложение. Разница в том, кто к вам входит.


Клиенты-физлица входят с Tinkoff ID. Для этого не обязательно быть клиентами Тинькофф — достаточно ввести свой номер телефона.

Клиенты-физлица заходят с Tinkoff ID
Партнеры — ИП и представители ООО — входят с Tinkoff ID для юрлиц. У них должен быть открыт расчетный счет в Тинькофф Бизнесе.
