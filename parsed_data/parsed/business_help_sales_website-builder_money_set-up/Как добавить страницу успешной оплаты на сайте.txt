Как добавить страницу успешной оплаты на сайте?

Клиент видит страницу успешной оплаты, когда оплачивает покупку на сайте. На этой странице можете рассказать, что клиенту надо знать после оплаты: например, сколько времени займет сборка и доставка заказа.


В конструкторе Тинькофф есть страница успеха по умолчанию — она появляется, если не настроили свою. Страница по умолчанию — это поп‑ап, то есть всплывающее окно.

Это страница успеха по умолчанию. Она появляется после успешной оплаты, если вы не добавляли свою страницу
Добавить свою страницу успеха можно только для формы заказа. Для формы заявки такую страницу настроить не получится. Как добавить форму заказа на сайт (https://www.tinkoff.ru/business/help/sales/website-builder/money/set-up/?card=add-form)

Чтобы создать страницу успеха:

- В конструкторе Тинькофф (https://business.tinkoff.ru/sites) выберите нужный сайт → «Добавить страницу» → «Пустой шаблон». Нажмите на значок плюса в начале страницы.- Откроется меню блоков. Для страницы успеха подойдут шаблоны из разделов «Текст», «Преимущества» и «О компании» — в них можно написать текст и добавить изображения.
- Заполните страницу, как вам нужно.
Можете изменить цвет и размер текста, выравнивание и способы выделения, добавить ссылку на другую страницу своего сайта
      - Нажмите «Опубликовать» справа вверху. Осталось настроить переход с платежной формы на страницу успеха.- Нажмите «Назад» слева вверху. Выберите из списка страницу, на которой есть форма заказа, и нажмите «Оплатить».
- Снизу появится меню настроек. Нажмите «Страница успеха» и выберите из списка свою страницу.
- Нажмите «Опубликовать» справа вверху, чтобы сохранить изменения.

Готово! Когда клиент оплатит товар или услугу на сайте, появится страница успеха.
