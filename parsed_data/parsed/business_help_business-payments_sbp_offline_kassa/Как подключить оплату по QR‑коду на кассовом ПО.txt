Как подключить оплату по QR‑коду на кассовом ПО?

Чтобы подключить оплату по QR‑коду на кассовом ПО:

- Подключите интернет-эквайринг от Тинькофф. Подать заявку  (https://www.tinkoff.ru/kassa/?internal_source=help-order)- Создайте магазин в личном кабинете Тинькофф Бизнеса на вкладке интернет-эквайринга. Пошаговая инструкция  (https://www.tinkoff.ru/business/help/business-payments/internet-acquiring/how-use/store/?card=q3)- Включите в созданном магазине прием платежей через СБП. Для этого перейдите на «Все сервисы» → «Интернет-эквайринг» → «Магазины» → нужный магазин → «Способы оплаты» → «Система быстрых платежей».
Вы увидите размер комиссии по платежам через СБП- Обратитесь к своим разработчикам для интеграции вашего кассового ПО с API Тинькофф. Раздел № 3 — как настроить взаимодействие с СБП через методы API (https://acdn.tinkoff.ru/static/documents/merchant_api_protocoI_eacq.pdf)- Передайте разработчикам уникальный код терминала (TerminalKey) и пароль (SecretKey), которые формируются после создания магазина, — они понадобятся при настройке интеграции. Где найти TerminalKey и SecretKey (/business/help/business-payments/sbp/offline/kassa/?card=q6)
В процессе интеграции можно выбрать способ вывода QR‑кода, например на пречеке, мониторе кассы или дополнительном дисплее. Все способы приема платежей по QR‑коду через кассу (/business/help/business-payments/sbp/offline/kassa/?card=q3)
Если у вас возникли вопросы, свяжитесь с персональным менеджером по телефону 8 800 700‑66‑66 (tel:8 800 700‑66‑66) или напишите в чат поддержки (https://tinkoffbusiness.onelink.me/1221431269/oocytv75) Тинькофф Бизнеса. Он поможет подключить интернет-эквайринг, создать магазин и включить оплату через СБП.
Перейти в чат поддержки можно с любой страницы личного кабинета Тинькофф Бизнеса — нажмите на желтую кнопку чата
