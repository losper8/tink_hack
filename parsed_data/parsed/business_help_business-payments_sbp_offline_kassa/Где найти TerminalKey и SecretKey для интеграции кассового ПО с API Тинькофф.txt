Где найти TerminalKey и SecretKey для интеграции кассового ПО с API Тинькофф?

TerminalKey и SecretKey указаны в личном кабинете Тинькофф Бизнеса в разделе «Интернет-эквайринг» → «Магазины» → нужный магазин → «Терминалы».

Значение TerminalKey — в строке «Терминал», а значение SecretKey — в строке «Пароль»
