Как добавить пункты самовывоза для одного города?

Чтобы добавить пункты самовывоза:

- В конструкторе Тинькофф (https://business.tinkoff.ru/sites/) выберите нужный сайт → «Настройки сайта» → «Доставка» → «Добавить».- В поле «Тип доставки» выберите «Самовывоз». В поле «Название» укажите общее название для всех пунктов самовывоза, например «Самовывоз в Москве», — оно будет отображаться у покупателей в корзине.
- Нажмите «Добавить адрес». Укажите адрес и название пункта выдачи, время его работы, способ до него добраться, срок доставки.

Передвиньте переключатель «Доступен» — он станет желтым. Тогда при оформлении заказа покупатель сможет выбрать это направление доставки.

Если нужно, укажите минимальную и максимальную стоимость заказа для выдачи в конкретном пункте. Для этого в разделе «Условия самовывоза» передвиньте переключатель «Платный» — он станет желтым. Заполните поля «Стоимость самовывоза», «Стоимость заказа для бесплатного самовывоза» и «Варианты оплаты».

Готово, изменения сохранятся автоматически. Теперь покупатели смогут выбрать нужный пункт самовывоза при оформлении заказа.
