Как добавить ключ от API Wildberries в Тинькофф Селлер?

Сначала нужно создать ключ на Wildberries:

- Зайдите в личный кабинет «WB Партнеры» (https://seller.wildberries.ru/supplier-settings/access-to-new-api) → Профиль → Настройки → Доступ к API.
-  Выберите все категории API и нажмите «Создать новый токен». Скопируйте токен. 
Не нажимайте «Только для чтения» при создании токена! В этом случае вы не сможете работать с Wildberries в Тинькофф Селлере.
- Зайдите в личный кабинет Тинькофф Бизнеса → «Селлер» → «Магазины». (https://business.tinkoff.ru/seller/markets) Если вы создаете новый магазин, вставьте скопированный токен в поле «Ключ».
При создании нового магазина вставьте скопированный токен в поле «Ключ»Если вы хотите обновить данные в существующем магазине, нажмите на его название. На карточке Wildberries нажмите на три точки в правом верхнем углу → «Изменить». Вставьте скопированный токен в оба поля для ключей.
- Нажмите кнопку «Добавить».
Готово, вы добавили ключ. Теперь вы можете работать с Wildberries в Тинькофф Селлере.
