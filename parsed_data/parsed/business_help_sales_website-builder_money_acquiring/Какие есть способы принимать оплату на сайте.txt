Какие есть способы принимать оплату на сайте?

Вы можете принимать оплату банковскими картами, через Tinkoff Pay, SberPay, Yandex Pay, Mir Pay и СБП. Оплата банковскими картами, через Tinkoff Pay и SberPay доступна сразу после подключения интернет-эквайринга, остальные способы можно включить в личном кабинете интернет-эквайринга.

- Зайдите в личный кабинет Тинькофф Бизнеса → «Интернет-эквайринг» → «Магазины». (https://business.tinkoff.ru/oplata/mpa/shops)
- Выберите ваш магазин → «Способы оплаты» → Tinkoff Pay или SberPay.
- Нажмите «Платежная форма Тинькофф» → «Настроить». Увидите, что способ оплаты подключен.
- Зайдите в личный кабинет Тинькофф Бизнеса → «Интернет-эквайринг» → «Магазины». (https://business.tinkoff.ru/oplata/mpa/shops)
- Выберите ваш магазин → «Способы оплаты» → Yandex Pay, Mir Pay или СБП.
- Нажмите «Платежная форма Тинькофф» → «Настроить» → «Включить».

Особенности способов оплатыБанковские карты Всем покупателям с российскими картами Visa, Mastercard и «Мир» Покупатели вводят данные карты и оплачивают покупку
Tinkoff Pay Клиентам Тинькофф Через платежную форму по кнопке «Оплатить с Tinkoff Pay» пользователь переходит в приложение Тинькофф
SberPay Клиентам Сбера Через платежную форму по кнопке «Оплатить с Tinkoff Pay» пользователь переходит в приложение Тинькофф
Mir Pay Держателям карты «Мир», которые пользуются смартфонами на Android Пользователь переходит по кнопке в приложение Mir Pay и в нем оплачивает покупку
Yandex Pay Пользователям сервисов Яндекса, которые авторизовались в Яндекс ID Пользователь нажимает на кнопку Yandex Pay, авторизуется в Яндекс ID и оплачивает картой, которая привязана к его аккаунту в Яндексе. Если карт несколько, выбирает нужную
Оплата через СБП Клиентам банков — участников СБП (https://sbp.nspk.ru/participants/) С компьютера — по кнопке «СБП» пользователь сканирует QR‑код, переходит в приложение банка и оплачивает покупку

С телефона — по кнопке «СБП» пользователь переходит в приложение банка и оплачивает покупку
