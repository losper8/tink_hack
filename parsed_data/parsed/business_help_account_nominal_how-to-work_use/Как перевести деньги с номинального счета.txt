Как перевести деньги с номинального счета?

Перевести деньги с номинального счета владелец может на основной расчетный счет, если это комиссия за услуги, либо бенефициарам.


Перевести деньги бенефициарам можно двумя способами:

- по реквизитам бенефициара;
- на карту бенефициара.


Принцип такой же, как при переводе с обычного расчетного счета. Только при заполнении платежного поручения вместо основного счета, с которого надо перевести деньги, выбирайте номинальный счет.

Как перевести деньги с расчетного счета в Тинькофф (/business/help/account/currency-ruble/transfer-and-withdraw/transfer/)
Комиссия за такие переводы устанавливается индивидуально при открытии номинального счета.


Переводами можно управлять автоматизированно через API. При подключении переводить деньги можно будет только через API для корректного учета оборота через номинальный счет. Документация Tinkoff API для работы с номинальным счетом (https://business.tinkoff.ru/openapi/sandbox/docs#tag/Nominalnye-scheta.-Beneficiary-i-rekvizity)
