Необходимо передавать номер заказа для использования HTTP‑нотификаций. Что делать?

Чтобы уведомления работали правильно, нужно передавать в заявке номер заказа — за это отвечает параметр orderNumber. Что такое параметр заявки orderNumber (https://forma.tinkoff.ru/docs/credit/help/methods/?type=api&method=create#orderNumber)

Вот как это сделать:

- Проверьте, что в вашем магазине подключены HTTP‑нотификации. Зайдите в личный кабинет руководителя → «Кредитование в магазинах» → выберите магазин → «Редактировать» → «Уведомления». Должна стоять галочка рядом с пунктом «По HTTP».- Проверьте с программистом параметры заявки и настройте передачу уникального номера для каждого заказа через orderNumber. Посмотреть методы скрипта и API (https://forma.tinkoff.ru/docs/credit/help/methods/)В разделе «Уведомления» вы можете выбрать один из двух или оба способа получать информацию о статусе заказа — по HTTP или e‑mail
Для уведомлений по почте передавать orderNumber не требуется.


Символ чата можно найти на каждой странице личного кабинета в нижнем правом углу
