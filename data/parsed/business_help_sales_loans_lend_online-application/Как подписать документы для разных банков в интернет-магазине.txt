## Как подписать документы для разных банков в интернет-магазине?
Есть несколько способов:
* с помощью СМС — удаленное подписание, доступно, если покупатель — клиент банка, в котором берет кредит или рассрочку;
* с помощью Self ID — удаленное подписание, доступно всем при оформлении кредита или рассрочки в Тинькофф Банке;
* полное подписание договора — покупатель должен лично встретиться с представителем или прийти в офис банка.
В каждом банке, подключенном к Тинькофф Кредит Брокеру, доступны свои способы подписания.
В Тинькофф
###### Способы подписания документов в Тинькофф
С помощью СМС
У покупателя нет продуктов Тинькофф
❌
Покупатель — клиент Тинькофф
✅ После одобрения заявки покупатель вводит СМС‑код в форме для подписания
Полное подписание договора
У покупателя нет продуктов Тинькофф
✅ Покупатель назначает встречу с представителем банка через форму заявки. Сотрудник приедет по указанному адресу и поможет подписать полный бумажный договор вручную
Покупатель — клиент Тинькофф
✅ Покупатель назначает встречу с представителем банка через форму заявки. Сотрудник приедет по указанному адресу и поможет подписать полный бумажный договор вручную
Self ID
У покупателя нет продуктов Тинькофф
✅ После одобрения покупателю нужно загрузить в форму заявки три фотографии:
* селфи с открытым паспортом на странице с фото;
* разворот паспорта с фото;
* разворот паспорта с регистрацией.
Затем покупатель вводит СМС‑код в форме для подписания
Покупатель — клиент Тинькофф
❌
В МТС Банке
###### Способы подписания документов в МТС Банке
С помощью СМС
У покупателя нет продуктов МТС Банка
❌
Покупатель — клиент МТС Банка
✅ После одобрения заявки покупатель загружает три фотографии:
* селфи;
* разворот паспорта с фото;
* разворот паспорта с регистрацией.
Затем покупатель вводит СМС‑код в форме для подписания
Полное подписание договора
У покупателя нет продуктов МТС Банка
✅ После оформления заявки покупатель получает СМС. До указанного срока покупателю нужно прийти в магазин МТС с паспортом и показать СМС‑сообщение сотруднику. Так покупатель подтвердит свою личность.
Покупатель — клиент МТС Банка
❌
В ОТП Банке
Удаленное подписание с помощью СМС. Доступно только для покупателей — клиентов ОТП Банка. После одобрения заявки покупатель вводит код из СМС прямо в форме для подписания.
В Ренессанс Кредит Банке
Удаленное подписание с помощью СМС. Доступно только для покупателей — клиентов Ренессанс Кредит Банка. После одобрения заявки покупатель вводит код из СМС прямо в форме для подписания.
Покупатель должен подписать заявку в день ее создания до 23:30 мск. Если этого не сделать, заявка архивируется и покупателю придется создавать новую.
В Почта Банке
Подписание в мобильном приложении Почта Банка или в любом отделении. Доступно только для покупателей — клиентов Почта Банка.
В любом отделении. Доступно для будущих покупателей — новых клиентов Почта Банка.
В Kviku Банке
Удаленное подписание с помощью СМС. Доступно только для покупателей — клиентов Kviku Банка. После одобрения заявки покупатель вводит код из СМС прямо в форме подписания.
В Совкомбанке
Удаленное подписание с помощью СМС. Доступно только для покупателей — клиентов Совкомбанка. После одобрения заявки покупатель вводит код из СМС прямо в форме для подписания.
