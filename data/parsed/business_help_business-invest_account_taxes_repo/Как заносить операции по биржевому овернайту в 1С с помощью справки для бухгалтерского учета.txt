## Как заносить операции по биржевому овернайту в 1С с помощью справки для бухгалтерского учета?
Сначала нужно создать договор в справочнике 1С и добавить в справочник тип актива. После этого можно будет создать шаблоны операций и использовать их в бухучете, меняя суммы в проводках и дату проведения операции.
Шаг 1. Подготовить 1С для работы с операциями по биржевому овернайту
1. В программе 1С в главном меню перейдите в раздел «Справочники» и в категории «Покупки и продажа» выберите «Контрагенты». ![](https://imgproxy.cdn-tinkoff.ru/compressed95:help_question_card_body_image_desktop_col_9/aHR0cHM6Ly9vcGlzLWNkbi50aW5rb2Zmam91cm5hbC5ydS9wbHV0by1iYWNrZW5kLWltYWdlcy9jMzBjOTcwNC4wNV9kZXNrdG9wLnBuZw==)
2. Создайте нового контрагента «АО НКО НКЦ» и внесите информацию как на скрине ниже → «Записать и закрыть». ![](https://imgproxy.cdn-tinkoff.ru/compressed95:help_question_card_body_image_desktop_col_9/aHR0cHM6Ly9vcGlzLWNkbi50aW5rb2Zmam91cm5hbC5ydS9wbHV0by1iYWNrZW5kLWltYWdlcy9jY2NhZDFlMy4wNl9kZXNrdG9wLnBuZw==)
3. В главном меню выберите раздел «Справочники» и в категории «Покупки и продажа» выберите «Договоры». ![](https://imgproxy.cdn-tinkoff.ru/compressed95:help_question_card_body_image_desktop_col_9/aHR0cHM6Ly9vcGlzLWNkbi50aW5rb2Zmam91cm5hbC5ydS9wbHV0by1iYWNrZW5kLWltYWdlcy9kNTlmZDA0NC4wN19kZXNrdG9wLnBuZw==)
4. Создайте «Договор об оказании услуг на финансовом рынке»: заполните номер договора брокерского обслуживания, укажите, что расчеты не облагаются НДС → «Записать и закрыть». ![](https://imgproxy.cdn-tinkoff.ru/compressed95:help_question_card_body_image_desktop_col_9/aHR0cHM6Ly9vcGlzLWNkbi50aW5rb2Zmam91cm5hbC5ydS9wbHV0by1iYWNrZW5kLWltYWdlcy9iYmE3YjcyNy4wMV9pbnZlc3Rfc21lLnBuZw==)
5. В справочнике появился новый договор по брокерскому обслуживанию. Его вы будете использовать для добавления информации в субконто по операциям в рамках брокерского счета.
6. Добавьте в справочник информацию по ценным бумагам КСУ: в главном меню выберите раздел «Справочники» → в строку поиска в верхней части экрана введите «Ценные бумаги» → выберите из выпадающего списка «Ценные бумаги» с припиской «Доходы и расходы» → «Создать». ![](https://imgproxy.cdn-tinkoff.ru/compressed95:help_question_card_body_image_desktop_col_9/aHR0cHM6Ly9vcGlzLWNkbi50aW5rb2Zmam91cm5hbC5ydS9wbHV0by1iYWNrZW5kLWltYWdlcy8yMjcwMjQ2Ni4wOV9kZXNrdG9wLnBuZw==)
7. Во всплывающем окне внесите наименование актива из поля «Сокращенное наименование актива» в брокерском отчете: для учета сделок по биржевому овернайту — «КСУ GC Bonds». Далее нажмите «Записать и закрыть». ![](https://imgproxy.cdn-tinkoff.ru/compressed95:help_question_card_body_image_desktop_col_9/aHR0cHM6Ly9vcGlzLWNkbi50aW5rb2Zmam91cm5hbC5ydS9wbHV0by1iYWNrZW5kLWltYWdlcy9kYTkyNzllZS4xMF9kZXNrdG9wLnBuZw==)
Готово! Теперь можно перейти к созданию шаблонов типовых операций.
Шаг 2. Создать шаблон типовой операции по первой части сделки биржевого овернайта в 1С
1. В главном меню выберите раздел «Операции» → «Типовые операции» → «Создать». ![](https://imgproxy.cdn-tinkoff.ru/compressed95:help_question_card_body_image_desktop_col_9/aHR0cHM6Ly9vcGlzLWNkbi50aW5rb2Zmam91cm5hbC5ydS9wbHV0by1iYWNrZW5kLWltYWdlcy9mZWYyNDJjMi4xMV9kZXNrdG9wLnBuZw==)
2. Заполните шаблон операции как на примере ниже — вам останется только изменить номер договора брокерского обслуживания, его можно посмотреть в брокерском отчете в шапке документа.
В проводке 91.02 Дб — 76.09 Кд в субконто выберите «Прочие внереализационные доходы», в параметре «Реализуемые активы» укажите тип данных «Ценная бумага», а в наименовании актива — ранее введенное «КСУ GC Bonds».
Суммы можно указать произвольные или вообще не указывать — на этом этапе вы создаете шаблон с проводками.
![](https://imgproxy.cdn-tinkoff.ru/compressed95:help_question_card_body_image_desktop_col_9/aHR0cHM6Ly9vcGlzLWNkbi50aW5rb2Zmam91cm5hbC5ydS9wbHV0by1iYWNrZW5kLWltYWdlcy84MTdlODM4OS4xMl9kZXNrdG9wLnBuZw==)
3. Нажмите «Записать и закрыть», чтобы сохранить шаблон типовой операции в реестре ручных операций 1С.
Шаг 3. Создать шаблон типовой операции по второй части сделки биржевого овернайта в 1С
1. В главном меню выберите раздел «Операции» → «Типовые операции» → «Создать». ![](https://imgproxy.cdn-tinkoff.ru/compressed95:help_question_card_body_image_desktop_col_9/aHR0cHM6Ly9vcGlzLWNkbi50aW5rb2Zmam91cm5hbC5ydS9wbHV0by1iYWNrZW5kLWltYWdlcy8zYjdmOThhMC4xM19kZXNrdG9wLnBuZw==)
2. Заполните шаблон операции как на примере ниже — вам останется только изменить номер договора брокерского обслуживания, его можно посмотреть в брокерском отчете в шапке документа.
В проводке 76.09 Дб — 91.01 Кд в субконто выберите «Прочие внереализационные доходы», в параметре «Реализуемые активы» укажите тип данных «Ценная бумага», а в наименовании актива — ранее введенное «КСУ GC Bonds».
Суммы можно указать произвольные или вообще не указывать — на этом этапе вы создаете шаблон с проводками.
![](https://imgproxy.cdn-tinkoff.ru/compressed95:help_question_card_body_image_desktop_col_9/aHR0cHM6Ly9vcGlzLWNkbi50aW5rb2Zmam91cm5hbC5ydS9wbHV0by1iYWNrZW5kLWltYWdlcy9hYjJhNmYyOS4xNF9kZXNrdG9wLnBuZw==)
Готово! Теперь шаблоны по операциям можно переиспользовать, чтобы вносить суммы по проводкам.
Если шаблоны созданы, можно использовать их для ведения бухгалтерского учета, подставляя нужные значения по каждой сделке биржевого овернайта. Эти значения можно брать из брокерского отчета.
Как делать проводки сделок биржевого овернайта по готовым шаблонам
1. В программе 1С в главном меню перейдите в раздел «Операции» и в категории «Бухгалтерский учет» выберите «Операции, введенные вручную». ![](https://imgproxy.cdn-tinkoff.ru/compressed95:help_question_card_body_image_desktop_col_9/aHR0cHM6Ly9vcGlzLWNkbi50aW5rb2Zmam91cm5hbC5ydS9wbHV0by1iYWNrZW5kLWltYWdlcy9lNTdmMDhhZC4xNV9kZXNrdG9wLnBuZw==)
2. Нажмите «Создать» → «Типовая операция» → выберите созданный ранее шаблон. ![](https://imgproxy.cdn-tinkoff.ru/compressed95:help_question_card_body_image_desktop_col_9/aHR0cHM6Ly9vcGlzLWNkbi50aW5rb2Zmam91cm5hbC5ydS9wbHV0by1iYWNrZW5kLWltYWdlcy8zZTg5MjBkOS4xNl9kZXNrdG9wLnBuZw==)
3. В окне типовой операции нажмите «Заполнить». ![](https://imgproxy.cdn-tinkoff.ru/compressed95:help_question_card_body_image_desktop_col_9/aHR0cHM6Ly9vcGlzLWNkbi50aW5rb2Zmam91cm5hbC5ydS9wbHV0by1iYWNrZW5kLWltYWdlcy85ZmUwOGY0Zi5pem9icmF6aGVuaWVfMjAyM18wN18yN18xNTI0MDI2MzYucG5n)
4. В подразделе «Бухгалтерский и налоговый учет» заполните столбец «Суммы». Нужные значения можно взять из брокерского отчета. [Как найти сделки по биржевому овернайту в брокерском отчете](/business/help/business-invest/account/taxes/repo/?card=q3)
Где в брокерском отчете брать сумму для проводок по первой части сделок РЕПО с КСУ
* Оплата КСУ с брокерского счета — из столбца «Количество» строки с видом сделки «РЕПО 1 Покупка»;
* получение КСУ — из столбца «Сумма сделки» строки с видом сделки «РЕПО 1 Покупка»;
* комиссия брокера — из столбца «Комиссия брокера» строки с видом сделки «РЕПО 1 Покупка».
Где в брокерском отчете брать сумму для проводок по второй части сделок РЕПО с КСУ
* Выплата на брокерский счет за КСУ — взять значение столбца «Сумма сделки» строки с видом сделки РЕПО 2 и вычесть из него значение столбца «Комиссия брокера» строки с видом сделки «РЕПО 1 Покупка»;
* возврат КСУ — из столбца «Количество» строки с видом сделки РЕПО 2;
* начисление дохода по сделке — взять значение столбца «Сумма сделки» строки с видом сделки РЕПО 2 и вычесть из него значение столбца «Сумма сделки» строки с видом сделки РЕПО 1.
![](https://imgproxy.cdn-tinkoff.ru/compressed95:help_question_card_body_image_desktop_col_9/aHR0cHM6Ly9vcGlzLWNkbi50aW5rb2Zmam91cm5hbC5ydS9wbHV0by1iYWNrZW5kLWltYWdlcy84ZmEwNTg5ZS5pem9icmF6aGVuaWVfMjAyM18wN18yN18xNTI1MDEwNDMucG5n)
5. Укажите дату и время сделки в поле даты — ее можно скопировать из столбцов «Дата заключения» и «Время» брокерского отчета.
![](https://imgproxy.cdn-tinkoff.ru/compressed95:help_question_card_body_image_desktop_col_9/aHR0cHM6Ly9vcGlzLWNkbi50aW5rb2Zmam91cm5hbC5ydS9wbHV0by1iYWNrZW5kLWltYWdlcy9hZDYzNjQ4My5pem9icmF6aGVuaWVfMjAyM18wN18yN18xNTI1MzU0NTQucG5n)
Отчет формируется в режиме Т-3 — информация по сделкам, которые вы совершили сегодня, отобразится в отчете через три рабочих дня.
6. Нажмите «Записать и закрыть».
