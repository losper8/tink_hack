## Какие ограничения есть в работе с N‑граммами в речевой аналитике?
В работе с N‑граммами есть ограничения, которые связаны с поиском:
* поиск N‑грамм недоступен для семантического и последовательного поиска — только для обычного; [Какие есть типы поиска](/business/help/solutions/speech-analytics-for-tinkoff/search/calls/?card=q2)
* поиск N‑грамм недоступен для чатов — только для звонков;
* если поиск звонков выдал больше 1000 релевантных звонков, речевая аналитика учтет N‑граммы только в первой тысяче;
* если вы хотите скачать Excel-таблицу с найденными в поиске звонками, N‑граммы в ней отражены не будут; [Как скачать таблицу](/business/help/solutions/speech-analytics-for-tinkoff/search/calls/?card=q16)
* при поиске N‑грамм нельзя использовать фильтры группы «Оценки»; [Какие есть группы фильтров для поиска звонков](/business/help/solutions/speech-analytics-for-tinkoff/search/calls/?card=q3)
* в результатах поиска речевая аналитика показывает до 500 биграмм и до 500 триграмм, которые используются чаще всего.
