Какие документы нужно предоставить?
Для предварительного решения по кредиту достаточно загрузить выписку по счетам из других банков.
Чтобы узнать окончательное решение по кредиту и получить деньги, нужно открыть счет, для этого понадобятся документы о регистрации бизнеса:
- для ИП обычно нужен только паспорт;
- для ООО — паспорт директора, приказ о его назначении и устав.

Иногда можем запросить документы, которые уже есть на руках. Например, бухгалтерский баланс или управленческий учет.
