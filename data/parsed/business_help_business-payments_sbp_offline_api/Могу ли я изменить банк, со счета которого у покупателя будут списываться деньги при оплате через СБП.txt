Могу ли я изменить банк, со счета которого у покупателя будут списываться деньги при оплате через СБП?
Это может настроить только покупатель. Как сделать Тинькофф банком по умолчанию для платежей по QR‑коду — инструкция для покупателя (/bank/help/payments/pay-for-purchases/sbp/qr-code/?card=q4)
Продавец не может выбирать, из какого банка он будет принимать оплату. Деньги спишутся со счета, который покупатель выбрал при оплате по QR‑коду, либо со счета, который вручную установлен приоритетным для оплаты через СБП.
