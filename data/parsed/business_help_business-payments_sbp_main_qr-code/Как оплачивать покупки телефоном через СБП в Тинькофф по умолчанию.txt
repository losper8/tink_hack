## Как оплачивать покупки телефоном через СБП в Тинькофф по умолчанию?
После настройки QR‑коды для оплаты через СБП будут открываться в приложении Тинькофф автоматически.
Чтобы сделать Тинькофф приложением для оплаты по QR‑коду через СБП, следуйте инструкции.
Для iOS
1. Откройте «Камеру» и наведите объектив на QR‑код.
2. В правом нижнем углу нажмите на значок QR‑кода и выберите Тинькофф. ![](https://imgproxy.cdn-tinkoff.ru/compressed95:help_question_card_body_image_desktop_col_9/aHR0cHM6Ly9vcGlzLWNkbi50aW5rb2Zmam91cm5hbC5ydS9wbHV0by1iYWNrZW5kLWltYWdlcy9hMmQzMDJkNS4wMl9zYnBfbmV3X2Rlc2tfNHgucG5n)Изменить приложение по умолчанию можно в любой момент при нажатии на кнопку QR-кода
Для Android
Если вы впервые на вашем устройстве сканируете QR‑код для оплаты через СБП, следуйте инструкции:
1. Откройте «Камеру» или другое приложение для сканирования QR‑кодов и отсканируйте QR‑код.
2. На всплывшем меню приложений, с помощью которых можно открыть ссылку, выберите Тинькофф и включите функцию «Открывать всегда».
Если на вашем устройстве после сканирования QR‑кода для оплаты через СБП открывается другое приложение, следуйте инструкции:
1. Нажмите «Настройки» → «Приложения» → «Приложения по умолчанию» → «Переход по ссылкам». ![](https://imgproxy.cdn-tinkoff.ru/compressed95:help_question_card_body_image_desktop_col_9/aHR0cHM6Ly9vcGlzLWNkbi50aW5rb2Zmam91cm5hbC5ydS9wbHV0by1iYWNrZW5kLWltYWdlcy83Yzk2YjY4My4wM19zYnBfbmV3X2Rlc2tfNHgucG5n)На разных устройствах и в разных версиях операционной системы путь к настройке приложений может отличаться
2. Выберите приложение Тинькофф и нажмите «Открывать ссылки в этом приложении» и выберите вариант «В этом приложении». Готово! Тинькофф станет приложением для оплаты через СБП по умолчанию. ![](https://imgproxy.cdn-tinkoff.ru/compressed95:help_question_card_body_image_desktop_col_9/aHR0cHM6Ly9vcGlzLWNkbi50aW5rb2Zmam91cm5hbC5ydS9wbHV0by1iYWNrZW5kLWltYWdlcy81NzY5ZTcwMy4wNF9zYnBfbmV3X2Rlc2tfNHgucG5n)Если путь к настройке приложений отличается на вашем устройстве, попробуйте воспользоваться справкой на сайте его производителя
На телефонах разных моделей и производителей способ настройки может отличаться. Если инструкция не подходит для вашей модели, попробуйте найти соответствующую инструкцию на сайте производителя.
