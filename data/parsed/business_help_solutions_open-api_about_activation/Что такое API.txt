## Что такое API
Application Programming Interface, API, — это инструмент, который помогает автоматизировать рутинные процессы в компании. С его помощью одно программа может взаимодействовать с другой — получать информацию или выполнять действия.
Например, компания работает с большим штатом сотрудников. С помощью интеграции бухгалтерской программы с банком через API бизнес может автоматизировать выплаты зарплат сотрудникам.
API можно разработать самостоятельно или воспользоваться готовым, если его предоставляют владельцы сервисов. Cвои API есть у банков, соцсетей, CRM‑систем. Им выгодно предоставлять другим компаниям доступ к своей информации. Например, так авиакомпании передают агрегаторам данные об авиабилетах и расширяют каналы продаж.
В Тинькофф Бизнесе тоже есть собственный API. Его можно использовать, чтобы интегрировать банковские сервисы в программы, которые использует бизнес. Например, автоматически обмениваться данными между CRM и личным кабинетом в Тинькофф.
Менеджеры смогут выставлять счета клиентам прямо из CRM, не заходя в личный кабинет банка. Это удобно для сотрудников и безопасно для бизнеса: руководителю не придется разграничивать доступ к финансам через банк.
API может воспользоваться любой клиент Тинькофф Бизнеса: базовые банковские функции бесплатны, а индивидуальные решения обсуждаем отдельно.
