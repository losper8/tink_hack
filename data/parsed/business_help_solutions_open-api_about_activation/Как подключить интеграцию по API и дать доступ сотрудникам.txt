## Как подключить интеграцию по API и дать доступ сотрудникам
Если у вас есть РКО в Тинькофф Бизнесе, [оставьте заявку на API](/business/open-api/?internal_source=blog-business) — мы ответим в течение двух рабочих дней и поможем с настройками.
Если планируете разработать интеграцию самостоятельно, ознакомьтесь [с нашей документацией для разработчиков.](https://developer.tinkoff.ru/docs/api/?internal_source=blog-business)
После того как вы интегрировали свои программы или сайт с сервисами Тинькофф, нужно предоставить сотрудникам доступ к новому интерфейсу. Уровни могут быть разные. Например, гендиректор может использовать все функции, а бухгалтер — только контролировать оплаты и поступления.
Доступ выдается с помощью защищенного токена — электронного ключа для авторизации пользователей. Можно выпустить неограниченное количество токенов с разными уровнями доступа.
Для выпуска токена авторизуйтесь в личном кабинете Тинькофф Бизнеса. Откройте меню «Сервисы» → «Интеграции» → «Интеграция с Тинькофф по API» → «Подключить».
![blog image](https://imgproxy.cdn-tinkoff.ru/compressed95:help_blog_image/aHR0cHM6Ly9vcGlzLWNkbi50aW5rb2Zmam91cm5hbC5ydS9wbHV0by1iYWNrZW5kLWJsb2ctaW1hZ2VzLzkwMDUxNTEwLjFfMi5wbmc=)
Выпустить токен можно только в личном кабинете: в мобильном приложении такой возможности нет
Затем введите IP‑адреса всех компьютеров, с которых будете использовать токен. Нажмите на стрелочку в строке «Выберите, к чему будет доступ через API». В выпадающем списке выберите операции, к которым хотите дать доступ по токену. Нажмите кнопку «Выпустить токен» → «Скопировать токен».
Токен действует год, но его можно досрочно отозвать в личном кабинете. Откройте меню «Управление профилем» → «Мой профиль» → «Безопасность» → «Приложения, у которых есть доступ к аккаунту». Выберите токен, который хотите удалить, нажмите на строку с ним и на кнопку «Удалить доступ».
