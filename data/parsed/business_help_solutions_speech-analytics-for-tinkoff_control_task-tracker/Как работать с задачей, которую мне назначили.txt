## Как работать с задачей, которую мне назначили?
Чтобы работать с задачами, которые вам назначали:
1. [Перейдите в «Развитие качества» → «Задачи» → «Все».](https://tqm.tinkoff.ru/qa/task-tracker/board/active)
2. Возьмите в работу задачу из колонки «Сделать». Статус «В работе» будет присвоен автоматически, как только вы выполните первое действие из задачи. Например, оцените звонок. Задача сразу получит статус «Сделано», если в ней будет только один звонок.
Вы можете вручную перевести задачу в работу. Для этого в карточке задачи нажмите на три точки → «Переместить в „В работе“».
![](https://imgproxy.cdn-tinkoff.ru/compressed95:help_question_card_body_image_desktop_col_9/aHR0cHM6Ly9vcGlzLWNkbi50aW5rb2Zmam91cm5hbC5ydS9wbHV0by1iYWNrZW5kLWltYWdlcy9mYTA4NGI5MS4yNy5wbmc=)Если в задаче несколько исполнителей, статус «В работе» появится, когда хотя бы один из исполнителей выполнит первое действие
Дальше работа над задачей будет зависеть от того, какой у нее тип. [Типы задач в таск‑трекере](/business/help/solutions/speech-analytics/control/task-tracker/?card=q3)
Как выполнить задачу с типом «Оценить коммуникацию»
Следуйте инструкции:
1. Откройте карточку задачи, чтобы увидеть список коммуникаций, которые нужно оценить. Нажмите на значок блокнота с галочкой, чтобы оценить коммуникацию. ![](https://imgproxy.cdn-tinkoff.ru/compressed95:help_question_card_body_image_desktop_col_9/aHR0cHM6Ly9vcGlzLWNkbi50aW5rb2Zmam91cm5hbC5ydS9wbHV0by1iYWNrZW5kLWltYWdlcy8yMTc2ODk4MC4yOC5wbmc=)Если в задаче несколько исполнителей и один из них уже отдал ее в работу, другой больше не сможет взять ее на оценку
2. Оцените коммуникацию по критериям оценочной формы. [Как оценить коммуникации](/business/help/solutions/speech-analytics/control/marks/?card=q3)
Если вы снизили балл или оставили комментарий, автоматически будет создана задача с типом «Обратная связь». Она будет назначена оператору, который работал с коммуникацией.
3. Если не можете оценить звонок, нажмите «Назад», чтобы вернуться к карточке задачи. Наведите курсор на звонок и выберите «Звонок не подходит».
![](https://imgproxy.cdn-tinkoff.ru/compressed95:help_question_card_body_image_desktop_col_9/aHR0cHM6Ly9vcGlzLWNkbi50aW5rb2Zmam91cm5hbC5ydS9wbHV0by1iYWNrZW5kLWltYWdlcy82NTk0ZjlhMC4yOS5wbmc=)
Откроется окно комментария. Напишите, почему не смогли оценить звонок, и нажмите «Отправить».
4. После оценки рядом со звонком появится зеленая галочка. Звонки, которые не получилось оценить, будут отмечены серым крестиком.
Повторите действия для остальных коммуникаций в задаче. Когда оцените все коммуникации, статус задачи изменится на «Сделано».
Как выполнить задачу с типом «Оценить оператора»
Следуйте инструкции:
1. Откройте карточку задачи, чтобы увидеть список операторов, чью работу нужно оценить.
2. В строке с оператором нажмите на значок блокнота с галочкой, чтобы оценить его работу. ![](https://imgproxy.cdn-tinkoff.ru/compressed95:help_question_card_body_image_desktop_col_9/aHR0cHM6Ly9vcGlzLWNkbi50aW5rb2Zmam91cm5hbC5ydS9wbHV0by1iYWNrZW5kLWltYWdlcy9jNzUzMzRkYS4yOC5wbmc=)Если в задаче несколько исполнителей и один из них уже начал оценивать оператора, другие не смогут взять его на оценку
3. Оцените оператора по критериям оценочной формы. [Как это сделать](/business/help/solutions/speech-analytics/control/marks/?card=q4)
Если вы снизили балл или оставили комментарий, задача с типом «Обратная связь» создастся автоматически. Она будет назначена оператору, которого вы оцениваете.
4. Если не можете оценить оператора, нажмите «Назад», чтобы вернуться в карточку задачи. Наведите курсор на строку с оператором и выберите «Оператор не подходит».
![](https://imgproxy.cdn-tinkoff.ru/compressed95:help_question_card_body_image_desktop_col_9/aHR0cHM6Ly9vcGlzLWNkbi50aW5rb2Zmam91cm5hbC5ydS9wbHV0by1iYWNrZW5kLWltYWdlcy9kZDViNzU3NS4zMC5wbmc=)
Откроется окно комментария. Напишите, почему не смогли оценить оператора, и нажмите «Отправить».
Рядом с теми операторами, которых вы оценили, будет стоять зеленая галочка. Операторы, которых не получилось оценить, будут отмечены серым крестиком.
Повторите действия для остальных операторов в задаче. Когда оцените всех операторов, статус задачи изменится на «Сделано».
Как выполнить задачу с типом «Обратная связь»
Следуйте инструкции:
1. Откройте карточку задачи, чтобы увидеть рекомендации по работе, которые вам оставил автор задачи, — они будут в поле «Задача». ![](https://imgproxy.cdn-tinkoff.ru/compressed95:help_question_card_body_image_desktop_col_9/aHR0cHM6Ly9vcGlzLWNkbi50aW5rb2Zmam91cm5hbC5ydS9wbHV0by1iYWNrZW5kLWltYWdlcy8xYzUwZjhhNC4xM19kZXNrdG9wLnBuZw==)
2. В блоке «Обратная связь» выберите одно из действий:
* «Соглашаюсь с обратной связью» — задача перейдет в статус «Сделано»;
* «Не соглашаюсь с обратной связью» — задача преобразуется в тип «Оспаривание».
![](https://imgproxy.cdn-tinkoff.ru/compressed95:help_question_card_body_image_desktop_col_9/aHR0cHM6Ly9vcGlzLWNkbi50aW5rb2Zmam91cm5hbC5ydS9wbHV0by1iYWNrZW5kLWltYWdlcy9iNTRlZGM0OC4xNF9kZXNrdG9wLnBuZw==)Свое решение можно дополнить комментарием — оценщик увидит его в истории работы над задачей
3. Нажмите «Перевести задачу». Тип задачи изменится на «Оспаривание», и она вернется автору. [С какими задачами можно работать в таск‑трекере](/business/help/solutions/speech-analytics/control/task-tracker/?card=q3)
Как выполнить задачу с типом «Оспаривание»
Следуйте инструкции:
1. Откройте карточку задачи. В блоке «Комментарии» отображается история обратной связи — согласие или оспаривание оценки. ![](https://imgproxy.cdn-tinkoff.ru/compressed95:help_question_card_body_image_desktop_col_9/aHR0cHM6Ly9vcGlzLWNkbi50aW5rb2Zmam91cm5hbC5ydS9wbHV0by1iYWNrZW5kLWltYWdlcy8zYzU2ZmY0MS4xNF81X2Rlc2t0b3AucG5n)
2. В блоке «Обратная связь» выберите одно из действий: «Соглашаюсь с оспариванием» или «Не соглашаюсь с оспариванием». Обоснуйте свое решение в блоке «Комментарий» и нажмите «Перевести задачу».
![](https://imgproxy.cdn-tinkoff.ru/compressed95:help_question_card_body_image_desktop_col_9/aHR0cHM6Ly9vcGlzLWNkbi50aW5rb2Zmam91cm5hbC5ydS9wbHV0by1iYWNrZW5kLWltYWdlcy9jNDEwOTI4MC4xNV9kZXNrdG9wLnBuZw==)
Если вы выбрали «Соглашаюсь с оспариванием», задача перейдет в статус «Сделано» и будет закрыта.
Если вы выбрали «Не соглашаюсь с оспариванием», задача меняет тип на «Оспаривание». Дальнейший процесс работы с задачей будет зависеть от того, к какому департаменту принадлежит оператор. [Процесс оспаривания для разных департаментов](https://wiki.tcsbank.ru/x/pOhBcQ)
Если вам не подходит процесс оспаривания в вашем департаменте, его можно поменять через запрос [в Forge.](https://forge.tcsbank.ru/)
При создании запроса используйте категорию «Назначить флоу оспаривания в таск трекере на отдел/группу». Назначить можно один из трех процессов, которые мы описали выше. Сделать новый нельзя.
