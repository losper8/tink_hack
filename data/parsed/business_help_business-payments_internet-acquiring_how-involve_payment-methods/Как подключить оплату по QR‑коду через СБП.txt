## Как подключить оплату по QR‑коду через СБП?
Подключить оплату по QR‑коду через СБП можно, только если у вас есть расчетный счет в Тинькофф Бизнесе. [Открыть счет](/business/account/?internal_source=help#form)
Это можно сделать в личном кабинете Тинькофф Бизнеса после того, как вы создали магазин и провели тестовые платежи. [Как и зачем тестировать прием оплаты](/business/help/business-payments/internet-acquiring/how-involve/integrate/?card=q16)
Подключение оплаты по QR‑коду через СБП зависит от двух условий:
* способа, как вы подключили интернет-эквайринг, — по CMS, платежному виджету, через API или SDK;
* того, где хотите настроить прием оплаты по QR‑коду, — через отдельную платежную кнопку или платежную форму, свою или Тинькофф Кассы. [Что такое платежная форма](/business/help/business-payments/internet-acquiring/how-involve/payment-form/?card=q1)
Вот как подключить СБП, если у вас:
Платежная форма Тинькофф Кассы, интеграция через платежный модуль или платежный виджет
Следуйте инструкции:
1. [Зайдите в личный кабинет Тинькофф Бизнеса → «Интернет-эквайринг» → «Магазины».](https://business.tinkoff.ru/oplata/mpa/shops)
2. Выберите ваш магазин → «Способы оплаты» → «СБП».
3. Нажмите «Платежная форма Тинькофф» → «Настроить» → «Включить».
Готово, клиенты смогут оплачивать покупки на вашем сайте по QR‑коду.

Своя платежная форма, интеграция по API или вы хотите настроить отдельную платежную кнопку
Следуйте инструкции:
1. [Зайдите в личный кабинет Тинькофф Бизнеса → «Интернет-эквайринг» → «Магазины».](https://business.tinkoff.ru/oplata/mpa/shops)
2. Выберите ваш магазин → «Способы оплаты» → «СБП» → «Своя платежная форма» → «Включить».
3. Для завершения настройки понадобится помощь разработчика, ему нужно следовать [инструкции.](/kassa/dev/payments/#tag/Oplata-cherez-SBP)
Интеграция по мобильному SDK
Следуйте инструкции:
1. [Зайдите в личный кабинет Тинькофф Бизнеса → «Интернет-эквайринг» → «Магазины».](https://business.tinkoff.ru/oplata/mpa/shops)
2. Выберите ваш магазин → «Способы оплаты» → «СБП» → «Приложение» → «Включить».
3. Для дальнейшей настройки понадобится помощь разработчика. Ему нужно следовать [инструкции.](/kassa/dev/payments/#tag/Oplata-cherez-SBP)
