## Как интегрировать сервис рассрочек по API?
Через разработчика. Передайте ему данные для входа в личный кабинет руководителя, параметры для интеграции и эти инструкции:
1. [Подключение сервиса рассрочек. ](https://forma.tinkoff.ru/docs/online/setup-types/service-setup)
2. [Подключение по API.](https://forma.tinkoff.ru/docs/online/setup-types/api/description-api)
3. [Настройки для работы.](https://forma.tinkoff.ru/docs/online/settings/auto-commit)
4. [Схема заявки.](https://forma.tinkoff.ru/docs/online/scheme/application)
Если в компании нет разработчика, можете выбрать подходящего из списка наших партнеров. Услуги разработчика платные. [Партнеры-интеграторы Тинькофф](/business/open-api/integrator/)
