## Какие фильтры нельзя использовать при создании тегов?
При создании тегов для обычного поиска звонков и чатов нельзя использовать фильтры:
* ID TQM;
* Внешний ID;
* С комментарием;
* Теги.
Для последовательного поиска звонков нельзя использовать фильтры:
* ID TQM;
* Внешний ID;
* Теги.
