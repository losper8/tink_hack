## С каким кассовым ПО у Тинькофф есть готовые модули интеграции?
У нас есть готовые модули интеграции для самых распространенных кассовых программ. Подробно о каждом ПО вас могут проконсультировать официальные поставщики — ниже даем ссылки на их сайты.
Эвотор
Сервис автоматизации для малого и среднего бизнеса. [Сайт Эвотор](https://evotor.ru/)
После интеграции QR‑код для оплаты можно выводить на экране кассового аппарата.
PDF
[Инструкция по интеграции оплаты по QR-коду в Эвотор](https://acdn.tinkoff.ru/static/pages/files/50f62910-b0c2-4db4-baf5-928be7e4f1a8.pdf)
Написать в поддержку сервиса можно [на его сайте.](https://evotor.ru/support/)
АТОЛ
Многофункциональное программное обеспечение для автоматизации торговли и логистики. [Сайт АТОЛ](https://www.atol.ru/)
После интеграции QR‑код для оплаты можно распечатать на пречеке.
PDF
[Инструкция по интеграции оплаты по QR-коду в Эвотор](https://acdn.tinkoff.ru/static/pages/files/37b2d2e5-0c0e-4a16-9712-07a8ec26b1db.pdf)
Связаться с поддержкой можно по телефону [8 495 137-56-66.](tel:84951375666)
1С
Сервис для автоматизации бухгалтерии — движения товаров и денег, расчета с клиентами, создания налоговых деклараций. [Сайт 1С](https://1c.ru/)
Подключить прием оплаты через Систему быстрых платежей можно в таких программах от 1С:
* «1С:Бухгалтерия государственного учреждения»;
* «1C:Бухгалтерия 8»;
* «1С:Гаражи»;
* «1С:Касса»;
* «1С:Комплексная автоматизация»;
* «1С:Мобильная касса»;
* «1С:РМК»;
* «1С:Розница 8»;
* «1С:Садовод»;
* «1С:УНФ»;
* «1C:Управление торговлей»;
* «1С:ERP Управление предприятием».
После интеграции QR‑код для оплаты можно выводить на экране терминала или печатать на пречеке.
[Как интегрировать оплату по QR‑коду в программах 1С](https://portal.1c.ru/applications/1C-SBP#additionalInfo)
Связаться с поддержкой можно по телефону [8 495 688‑10‑01](tel:8 495 688-10-01) или [8 495 956‑11‑81.](tel:8 495 956-11-81)
ШТРИХ-М
Сервис для автоматизации бизнеса, комплексных решений для транспорта и организации городского пространства, социальной сферы. [Сайт ШТРИХ‑М](https://www.shtrih-m.ru/)
После интеграции можно генерировать динамический QR‑код СБП, выводить его на экран кассы, дисплей покупателя или печатать на пречеке.

Lifepay
Сервис для автоматизации и развития торговли, учета, кассовых операций. Есть готовые решения для эл. коммерции, доставки, туризма, транспорта. [Сайт Lifepay](https://life-pay.ru/)
После интеграции QR‑код для оплаты можно выводить на смарт-терминале.
Чтобы интегрировать сервис, обратитесь [в его поддержку на сайте.](https://life-pay.ru/company/support/#)
БИФИТ КАССА
Кассовое ПО и ПО автоматизации для разных видов бизнеса: розницы, курьерской доставки, общепита и транспорта. [Сайт БИФИТ КАССЫ](https://kassa.bifit.com/)
После интеграции сможете принимать платежи через СБП двумя способами:
* выводить QR‑код на экран кассы, дополнительный дисплей или чек;
* отправлять ссылку на оплату в мессенджеры, на эл. почту или по СМС.
[Как интегрировать оплату по QR‑коду в БИФИТ КАССУ](https://kassa.bifit.com/wiki/index.php?title=%D0%9F%D0%BE%D0%B4%D0%BA%D0%BB%D1%8E%D1%87%D0%B5%D0%BD%D0%B8%D0%B5_%D0%A1%D0%91%D0%9F_%D0%BE%D1%82_%D0%9D%D0%A1%D0%9F%D0%9A)
Связаться с поддержкой можно по телефону [8 499 704-06-98.](tel:8 499 704-06-98)
iiko и мобильное приложение iikoWaiter
iiko — программа для автоматизации любых видов ресторанного бизнеса. [Сайт iiko](https://iiko.ru/)
iikoWaiter — приложение для смартфона или планшета на iOS и Android, которое позволяет официантам фиксировать заказы, передавать их на кухню и принимать оплату. [Сайт iikoWaiter](https://iikowaiter.ru/)
После интеграции можно выводить QR‑код любым удобным способом:
* в виде платежной формы на сайте и в любом другом интерфейсе, например на втором экране кассы;
* в мобильном приложении iikoWaiter;
* печатать коды на чеке или пречеке.
[Как интегрировать оплату по QR‑коду в iiko](https://ru.iiko.help/articles/#!special-iiko/tinkoff-bank/q/%25D1%2581%25D0%25B1%25D0%25BF/qid/1055791/qp/2)
Написать в поддержку можно по эл. почте [support@iikowaiter.ru.](mailto:support@iikowaiter.ru)
R_keeper и мобильная версия r_keeper Waiter 5
R_keeper — программа для автоматизации ресторанного бизнеса. [Сайт r_keeper](https://rkeeper.ru/)
R‑keeper Waiter 5 — приложение для смартфона или планшета на iOS и Android, которое позволяет официантам фиксировать заказы, передавать их на кухню и принимать оплату. Устанавливается после настройки основного ПО r_keeper.
После интеграции можно печатать QR‑код на чеке и выводить на дополнительный дисплей. Через r‑keeper Waiter 5 официанты смогут принимать оплату по QR‑коду на экране смартфона.
[Как интегрировать оплату по QR‑коду в r_keeper](https://docs.rkeeper.ru/rk7/latest/ru/sbp-dlya-tin-koff-19612188.html)
Написать в поддержку можно по эл. почте [integrations@rkeeper.ru.](mailto:integrations@rkeeper.ru)

СБИС и СБИС «Розница»
СБИС — система электронного документооборота и отчетности. [Сайт СБИС](https://sbis.ru/)
СБИС «Розница» — офлайн-приложение на Windows для приема платежей и управления программой лояльности в магазинах.
После интеграции продавцы смогут принимать оплату через СБП несколькими способами:
* печатать QR‑код на пречеке;
* выводить QR‑код на экран кассы;
* добавить виджет на сайт или в соцсети с вариантом оплаты через СБП;
* выставить покупателю счет с QR‑кодом по эл. почте;
* отправить ссылку на оплату через СБП по СМС или эл. почте.
[Как интегрировать оплату по QR‑коду в СБИС](https://sbis.ru/help/account/acquiring/qr/qr-payment-tinkoff?st=2)
Связаться с поддержкой можно по телефону [8 495 123-34-07.](tel:84951233407)
