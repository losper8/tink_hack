Как создать и настроить оценочную форму для автооценки?

Форму с настроенной автооценкой можно использовать и для ручных оценок. Удобно, если часть пунктов из формы возможно оценить только вручную.


В разделе «Администрирование»:

- Перейдите в раздел «Формы» → «Создать новую форму».- Пропустите поле «Мотивация» — его не нужно заполнять.- 
Включите тумблер «Автооценка» — он расположен под блоком «Мотивация».

Проверьте, что тумблер «Можно не проставлять баллы при заполнении формы» отключен — он должен быть серого цвета. Иначе баллы начисляться не будут.
- Под тумблером «Автооценка» выберите коммуникации, которые будет оценивать речевая аналитика — звонки или чаты.- Нажмите «Добавить вопросы» — вы попадете во второй раздел настройки оценочной формы. Именно тут вы создаете и настраиваете критерии оценки.

Все вопросы оценочной формы можно распределить по тематическим блокам. В каждом блоке может быть неограниченное число вопросов.

Например, вам нужно, чтобы речевая аналитика во время оценки звонков проверяла, насколько операторы вежливы и как они соблюдают скрипты.

В этом случае можно создать два тематических блока. Первый — «Вежливость оператора» с вопросами для оценки «Как оператор поздоровался» и «Как оператор попрощался».

Второй блок — «Соблюдение скриптов». В него можно включить вопрос для оценки «Как оператор продавал товар по скрипту».
- Настройте первый блок вопросов. Для этого введите его название в соответствующую пустую строку.
В нашем примере первый блок называется «Вежливость оператора»-  Добавьте в первый блок вопросы, по которым речевая аналитика будет оценивать коммуникации. Каждый вопрос настраивается отдельно.

Чтобы создать вопрос, нажмите на значок плюса слева от названия блока и выберите тип вопроса «Несколько вариантов». Автооценка работает только с этим типом вопросов.
Речевая аналитика будет «отвечать» на вопросы оценочной формы, начислять, отнимать или не проставлять баллы — зависит от того, какие фильтры и условия получения баллов выберете- Заполните строки «Введите вопрос» и «Введите ответ».
Чтобы добавить новый вариант ответа к вопросу, нажмите на значок плюса слева от существующего ответа → «Новый вариант» - Теперь к каждому ответу нужно настроить критерии, по которым речевая аналитика будет оценивать коммуникации.

Кликните на строку ответа. Справа появится блок настроек ответа. Под списком параметров нажмите «Автооценка» — откроются поля для ввода речевых фильтров.
Можно нажать на слово «Автооценка» или на круглый значок справа от него
В выпадающем списке выберите фильтры, которые соответствуют вашему вопросу и ответу. Заполните их значения.

Например, в вопросе «Как оператор поздоровался» для ответа «Вежливо» мы добавим фильтр «Оператор произнес». В значении фильтра укажем «Словарь приветственных слов». Подробнее про словари (https://www.tinkoff.ru/business/help/solutions/speech-analytics/setup/dictionaries/)
Это значит, что речевая аналитика будет прослушивать звонки и искать в них соответствующие слова из словаря. Если операторы их произносили, в автооценке на вопрос «Как оператор поздоровался» речевая аналитика выберет вариант «Вежливо».

Для автооценки звонков доступны фильтры групп «Речевая аналитика» и «Параметры звонка», за исключением фильтров «С комментарием», «ID в TQM» и «Внешний ID». Подробнее про фильтры для звонков (https://www.tinkoff.ru/business/help/solutions/speech-analytics/search/calls/?card=q3)
Для автооценки чатов доступны фильтры групп «Речевая аналитика» и «Параметры чата», за исключением фильтров «С комментарием», «ID в TQM» и «Внешний ID». Подробнее про фильтры для чатов (https://www.tinkoff.ru/business/help/solutions/speech-analytics/search/chats/?card=q2)
Чтобы автооценка работала корректно, нужно настроить критерии для каждого ответа в вопросе. Для этого кликните на следующий вариант ответа и повторите действия из 8 шага.
- Проверьте, включен ли тумблер «Проставлять балл», чтобы указать баллы автооценки за ответ. Если нет, нажмите на тумблер — он станет желтым.
- В каждом ответе укажите, сколько баллов оператор получит по автооценке. Задайте отрицательное число, чтобы вычесть баллы — например, за грубые высказывания.

Что важно знать о начислении баллов в автооценке:
- баллы за ответ будут начисляться, только если все выбранные фильтры для него применимы к коммуникации;- баллы за ответ не будут начисляться, если не укажете их.- Если нужно, добавьте новый вопрос. Для этого снова нажмите на значок плюса слева от названия блока и выберите тип вопроса «Несколько вариантов». Повторите действия из 6 шага.
В нашем примере мы создали два вопроса в блоке «Вежливость оператора» — как оператор поздоровался и попрощался- 
Если нужно, добавьте новый блок вопросов. Для этого снова нажмите на значок плюса слева от названия существующего блока и выберите «Новый блок». Повторите действия из 5 шага.

По умолчанию все коммуникации подходят для автооценки. После создания оценочной формы все новые коммуникации будут автоматически оценены по ней.


Можно ограничить коммуникации, доступные для автооценки. Для этого следуйте инструкции:

- Нажмите «Настройки формы» → «Автооценка».
- В выпадающем списке выберите фильтры, по которым хотите ограничить коммуникации. Задайте нужные значения.

Например, вам нужно, чтобы речевая аналитика выставляла автооценку только звонкам, которые длились от 7 до 15 минут. Выберите фильтр «Длительность звонка» и задайте для него значения в формате чч:мм:сс.
- Настройки автооценки в оценочной форме завершены. Сохраните оценочную форму. Как это сделать (https://www.tinkoff.ru/business/help/solutions/speech-analytics/setup/create-forms/?card=q7)
