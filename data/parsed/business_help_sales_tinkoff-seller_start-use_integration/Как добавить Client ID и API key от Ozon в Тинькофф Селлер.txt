## Как добавить Client ID и API key от Ozon в Тинькофф Селлер?
Скопируйте Client ID и API key в личном кабинете Ozon и добавьте в Тинькофф Селлер. Вот как это сделать:
1. [Зайдите в личный кабинет Ozon Seller, нажмите на иконку профиля → «Настройки» → «API‑ключи».](https://seller.ozon.ru/app/settings/api-keys?currentTab=sellerApi) Скопируйте Client ID.
2. Чтобы получить API key, нажмите «Сгенерировать ключ».
3. В поле «Название ключа» введите понятное для вас название. Например, «Тинькофф Селлер», если создаете ключ для интеграции с Тинькофф Селлером.
В блоке «Типы токена» поставьте галочку напротив пункта Admin. Нажмите «Сгенерировать».
4. Скопируйте API key. Будьте внимательны: увидеть и скопировать ключ доступа можно только при его создании. Если не скопируете ключ сразу, придется создавать новый.
5. [Зайдите в личный кабинет Тинькофф Бизнеса → «Селлер» → «Магазины» → «Добавить магазин»](https://business.tinkoff.ru/seller/markets/add) → Ozon.
6. Вставьте Client ID и API key в соответствующие поля и нажмите «Добавить».
Готово! Вы добавили Client ID и API key от Ozon в Тинькофф Селлер.
