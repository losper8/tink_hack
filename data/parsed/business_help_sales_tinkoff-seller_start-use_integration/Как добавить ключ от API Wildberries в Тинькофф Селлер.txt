## Как добавить ключ от API Wildberries в Тинькофф Селлер?
Сначала нужно создать ключ на Wildberries:
1. [Зайдите в личный кабинет «WB Партнеры»](https://seller.wildberries.ru/supplier-settings/access-to-new-api) → Профиль → Настройки → Доступ к API.
2. Выберите все категории API и нажмите «Создать новый токен». Скопируйте токен.
Не нажимайте «Только для чтения» при создании токена! В этом случае вы не сможете работать с Wildberries в Тинькофф Селлере.
3. [Зайдите в личный кабинет Тинькофф Бизнеса → «Селлер» → «Магазины».](https://business.tinkoff.ru/seller/markets) Если вы создаете новый магазин, вставьте скопированный токен в поле «Ключ». ![](https://imgproxy.cdn-tinkoff.ru/compressed95:help_question_card_body_image_desktop_col_9/aHR0cHM6Ly9vcGlzLWNkbi50aW5rb2Zmam91cm5hbC5ydS9wbHV0by1iYWNrZW5kLWltYWdlcy85N2M3MmE0Ni4wMV9kZXNrdG9wXzE3LnBuZw==)При создании нового магазина вставьте скопированный токен в поле «Ключ»
Если вы хотите обновить данные в существующем магазине, нажмите на его название. На карточке Wildberries нажмите на три точки в правом верхнем углу → «Изменить». Вставьте скопированный токен в оба поля для ключей.
4. Нажмите кнопку «Добавить».
Готово, вы добавили ключ. Теперь вы можете работать с Wildberries в Тинькофф Селлере.
