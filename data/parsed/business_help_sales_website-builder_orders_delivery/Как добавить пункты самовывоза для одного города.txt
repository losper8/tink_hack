## Как добавить пункты самовывоза для одного города?
Чтобы добавить пункты самовывоза:
1. [В конструкторе Тинькофф](https://business.tinkoff.ru/sites/) выберите нужный сайт → «Настройки сайта» → «Доставка» → «Добавить».
2. В поле «Тип доставки» выберите «Самовывоз». В поле «Название» укажите общее название для всех пунктов самовывоза, например «Самовывоз в Москве», — оно будет отображаться у покупателей в корзине. ![](https://imgproxy.cdn-tinkoff.ru/compressed95:help_question_card_body_image_desktop_col_9/aHR0cHM6Ly9vcGlzLWNkbi50aW5rb2Zmam91cm5hbC5ydS9wbHV0by1iYWNrZW5kLWltYWdlcy8wZmIzOTFiMy5kZXNrdG9wXzMzLnBuZw==)
3. Нажмите «Добавить адрес». Укажите адрес и название пункта выдачи, время его работы, способ до него добраться, срок доставки.
Передвиньте переключатель «Доступен» — он станет желтым. Тогда при оформлении заказа покупатель сможет выбрать это направление доставки.
![](https://imgproxy.cdn-tinkoff.ru/compressed95:help_question_card_body_image_desktop_col_9/aHR0cHM6Ly9vcGlzLWNkbi50aW5rb2Zmam91cm5hbC5ydS9wbHV0by1iYWNrZW5kLWltYWdlcy8xYzcwYTY2Ny5kZXNrdG9wXzM0LnBuZw==)
Если нужно, укажите минимальную и максимальную стоимость заказа для выдачи в конкретном пункте. Для этого в разделе «Условия самовывоза» передвиньте переключатель «Платный» — он станет желтым. Заполните поля «Стоимость самовывоза», «Стоимость заказа для бесплатного самовывоза» и «Варианты оплаты».
![](https://imgproxy.cdn-tinkoff.ru/compressed95:help_question_card_body_image_desktop_col_9/aHR0cHM6Ly9vcGlzLWNkbi50aW5rb2Zmam91cm5hbC5ydS9wbHV0by1iYWNrZW5kLWltYWdlcy8yNWZiMmY3My5kZXNrdG9wXzM1LnBuZw==)
Готово, изменения сохранятся автоматически. Теперь покупатели смогут выбрать нужный пункт самовывоза при оформлении заказа.
Добавьте по аналогии остальные пункты самовывоза для конкретного города.
