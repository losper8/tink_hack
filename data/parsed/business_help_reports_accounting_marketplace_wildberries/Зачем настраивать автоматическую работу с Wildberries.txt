## Зачем настраивать автоматическую работу с Wildberries?
Интеграция по API с Wildberries упрощает работу с отчетностью. Не нужно рассчитывать комиссию маркетплейса и загружать отчеты вручную: достаточно ввести API‑ключ в личном кабинете Бухгалтерии — и все операции Wildberries будут считаться автоматически и корректно. [Как подключить API‑интеграцию](/business/help/reports/tinkoff-accounting/online-accounting/wildberries/?card=q14)
Загруженные по API‑интеграции отчеты не нужно скачивать на компьютер. Суммы по операциям за рассчитанный период отображаются в ленте операций.
Вот в чем преимущества интеграции по API с Wildberries.
Экономия времени. Работа с отчетностью после подключения API‑интеграции будет занимать несколько минут в квартал. Это быстрее, чем учитывать все операции и самостоятельно загружать отчеты.
Экономия денег. Для работы с отчетами не придется нанимать бухгалтера. Подключить интеграцию по API можно бесплатно в мобильном приложении или в личном кабинете Тинькофф Бизнеса.
Отсутствие штрафов. При ручном расчете комиссии можно совершить ошибку и отправить неверные данные в налоговую. В таком случае вам выпишут штраф. Интеграция по API поможет избежать штрафов: система безошибочно рассчитывает суммы за операции.
Доступ к базовой версии Тинькофф Селлера. Клиенты, подключившие интеграцию по API, могут использовать сервис для работы с маркетплейсами — Тинькофф Селлер. В нем можно собирать аналитику по продажам, редактировать цену товаров, вести учет остатков по складам и отвечать на отзывы покупателей. [Что такое Тинькофф Селлер](/business/help/sales/tinkoff-seller/about/what-is-it/)
