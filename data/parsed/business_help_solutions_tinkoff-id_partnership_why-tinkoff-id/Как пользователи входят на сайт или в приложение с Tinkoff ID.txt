Как пользователи входят на сайт или в приложение с Tinkoff ID?

С помощью Tinkoff ID легко входить на сайт или в приложение. Входить могут обычные и корпоративные пользователи — ИП и представители ООО.

Физлица входят через Tinkoff ID. Чтобы входить на сайты и приложения, не обязательно быть клиентом Тинькофф — достаточно указать номер телефона. Зачем физлицам входить на сайты и в приложения с Tinkoff ID (https://www.tinkoff.ru/bank/help/general/tinkoff-id/customer/how-to-use/?card=q1)
Корпоративные пользователи входят через Tinkoff ID для юрлиц и должны иметь расчетный счет в Тинькофф Бизнесе.


Порядок входа одинаковый для всех.

Как пользователь входит на сайт- Нажимает на кнопку «Войти с Тинькофф» и вводит номер телефона.
Пользователю не нужно указывать e-mail или пароль- Подтверждает вход по СМС.
- Если у пользователя нет счета в Тинькофф, вы можете попросить его ввести дополнительные данные, например эл. почту. После этого пользователь входит на сайт, его данные автоматически сохраняются в профиле. Если дополнительные данные не нужны, пользователь входит на сайт сразу после подтверждения телефона.

Если пользователь — клиент Тинькофф, он видит окно со списком данных, которые получит сайт, и соглашается на передачу данных. После этого пользователь возвращается авторизованным на ваш сайт, его данные автоматически сохраняются в профиле.
Данные запрашиваются при первом входе на сайтКак пользователь входит в приложение- Если у пользователя нет счета в Тинькофф: нажимает на кнопку «Войти с Тинькофф», вводит номер телефона и подтверждает его по СМС.

Если пользователь — клиент Тинькофф: нажимает на кнопку «Войти с Тинькофф», затем подтверждает вход через мобильное приложение Тинькофф — по коду, отпечатку пальца или Face ID.
В этом примере пользователь заходит по Face ID, ему не нужно указывать e-mail или пароль- Если у пользователя нет счета в Тинькофф: приложение может запросить дополнительные данные, например эл. почту. После этого пользователь входит в приложение, его данные автоматически сохраняются в профиле. Если дополнительные данные не нужны, авторизация происходит сразу после подтверждения телефона.

Если пользователь — клиент Тинькофф: он видит окно со списком данных, которые получит приложение, и соглашается на передачу данных. После этого пользователь входит в приложение, его данные автоматически сохраняются в профиле.
