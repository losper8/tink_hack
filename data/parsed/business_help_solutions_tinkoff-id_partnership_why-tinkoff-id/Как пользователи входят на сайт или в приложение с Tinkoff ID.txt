## Как пользователи входят на сайт или в приложение с Tinkoff ID?
С помощью Tinkoff ID легко входить на сайт или в приложение. Входить могут обычные и корпоративные пользователи — ИП и представители ООО.
Физлица входят через Tinkoff ID. Чтобы входить на сайты и приложения, не обязательно быть клиентом Тинькофф — достаточно указать номер телефона. [Зачем физлицам входить на сайты и в приложения с Tinkoff ID](/bank/help/general/tinkoff-id/customer/how-to-use/?card=q1)
Корпоративные пользователи входят через Tinkoff ID для юрлиц и должны иметь расчетный счет в Тинькофф Бизнесе.
Порядок входа одинаковый для всех.
Как пользователь входит на сайт
1. Нажимает на кнопку «Войти с Тинькофф» и вводит номер телефона. ![](https://imgproxy.cdn-tinkoff.ru/compressed95:help_question_card_body_image_desktop_col_9/aHR0cHM6Ly9vcGlzLWNkbi50aW5rb2Zmam91cm5hbC5ydS9wbHV0by1iYWNrZW5kLWltYWdlcy8yZjlhYmIwMy4wMV90aW5rb2ZmX2lkX3ByYXZraV9kZXNrdG9wXzR4LnBuZw==)Пользователю не нужно указывать e-mail или пароль
2. Подтверждает вход по СМС.
3. Если у пользователя нет счета в Тинькофф, вы можете попросить его ввести дополнительные данные, например эл. почту. После этого пользователь входит на сайт, его данные автоматически сохраняются в профиле. Если дополнительные данные не нужны, пользователь входит на сайт сразу после подтверждения телефона.
Если пользователь — клиент Тинькофф, он видит окно со списком данных, которые получит сайт, и соглашается на передачу данных. После этого пользователь возвращается авторизованным на ваш сайт, его данные автоматически сохраняются в профиле.
![](https://imgproxy.cdn-tinkoff.ru/compressed95:help_question_card_body_image_desktop_col_9/aHR0cHM6Ly9vcGlzLWNkbi50aW5rb2Zmam91cm5hbC5ydS9wbHV0by1iYWNrZW5kLWltYWdlcy82NjNlMWE5Zi4wMl90aW5rb2ZmX2lkX3ByYXZraV9kZXNrdG9wXzR4LnBuZw==)Данные запрашиваются при первом входе на сайт
Как пользователь входит в приложение
1. Если у пользователя нет счета в Тинькофф: нажимает на кнопку «Войти с Тинькофф», вводит номер телефона и подтверждает его по СМС.
Если пользователь — клиент Тинькофф: нажимает на кнопку «Войти с Тинькофф», затем подтверждает вход через мобильное приложение Тинькофф — по коду, отпечатку пальца или Face ID.
![](https://imgproxy.cdn-tinkoff.ru/compressed95:help_question_card_body_image_desktop_col_9/aHR0cHM6Ly9vcGlzLWNkbi50aW5rb2Zmam91cm5hbC5ydS9wbHV0by1iYWNrZW5kLWltYWdlcy8xMjkyYWVmMS4wM190aW5rb2ZmX2lkX3ByYXZraV9kZXNrdG9wXzR4LnBuZw==)В этом примере пользователь заходит по Face ID, ему не нужно указывать e-mail или пароль
2. Если у пользователя нет счета в Тинькофф: приложение может запросить дополнительные данные, например эл. почту. После этого пользователь входит в приложение, его данные автоматически сохраняются в профиле. Если дополнительные данные не нужны, авторизация происходит сразу после подтверждения телефона.
Если пользователь — клиент Тинькофф: он видит окно со списком данных, которые получит приложение, и соглашается на передачу данных. После этого пользователь входит в приложение, его данные автоматически сохраняются в профиле.
