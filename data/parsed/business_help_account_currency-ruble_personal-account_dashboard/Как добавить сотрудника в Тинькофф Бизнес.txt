## Как добавить сотрудника в Тинькофф Бизнес?
Это можно сделать в мобильном приложении Тинькофф Бизнеса и личном кабинете на сайте [tinkoff.ru](/), если у сотрудника есть гражданство РФ. Если гражданства нет, напишите [в чат поддержки](https://tinkoffbusiness.onelink.me/1221431269/oocytv75) Тинькофф Бизнеса или позвоните по телефону [8 800 700‑66‑66.](tel:8 800 700‑66‑66) Менеджер подскажет, как добавить пользователя и какие документы нужно собрать для оформления.
С телефона — в приложении Тинькофф Бизнеса
1. Откройте «Еще» → «Сотрудники» на iPhone или «Пользователи» на Android → «Добавить сотрудника / пользователя».
2. Выберите «РФ» → «Дальше». ![](https://imgproxy.cdn-tinkoff.ru/compressed95:help_question_card_body_image_desktop_col_9/aHR0cHM6Ly9vcGlzLWNkbi50aW5rb2Zmam91cm5hbC5ydS9wbHV0by1iYWNrZW5kLWltYWdlcy9jYWZlMmIwOC4wMV9kZXNrdG9wLnBuZw==)
3. Введите номер телефона и ФИО пользователя → «Дальше» → «Доступ к личному кабинету» и выберите уровень доступа. ![](https://imgproxy.cdn-tinkoff.ru/compressed95:help_question_card_body_image_desktop_col_9/aHR0cHM6Ly9vcGlzLWNkbi50aW5rb2Zmam91cm5hbC5ydS9wbHV0by1iYWNrZW5kLWltYWdlcy8wODU1NjZjYS5kZXNrdG9wXzIucG5n)
«С правом подписи» — соответствует уровню «Подписант». Для оформления понадобятся паспортные данные пользователя. Мы подготовим документы, чтобы выпустить электронную подпись.
«Без права подписи» — соответствует уровню «Менеджер». Для оформления не понадобятся паспортные данные пользователя. Он сможет смотреть информацию по счетам, создавать платежи и отправлять их на подпись.
[Подробнее про уровни доступа](/business/help/account/currency-ruble/personal-account/dashboard/?card=q7)
4. Введите код подтверждения из СМС. Готово! Вы добавили пользователя в Тинькофф Бизнес.
С телефона — в личном кабинете на сайте tinkoff.ru
1. Вверху экрана нажмите на название компании или ИП → «Пользователи». ![](https://imgproxy.cdn-tinkoff.ru/compressed95:help_question_card_body_image_desktop_col_9/aHR0cHM6Ly9vcGlzLWNkbi50aW5rb2Zmam91cm5hbC5ydS9wbHV0by1iYWNrZW5kLWltYWdlcy81OWMyMWFiMC4wMl8wM19kZXNrdG9wLnBuZw==)
2. Нажмите «Все пользователи» → «Добавить» → «РФ» → «Дальше». ![](https://imgproxy.cdn-tinkoff.ru/compressed95:help_question_card_body_image_desktop_col_9/aHR0cHM6Ly9vcGlzLWNkbi50aW5rb2Zmam91cm5hbC5ydS9wbHV0by1iYWNrZW5kLWltYWdlcy9mNmI3YjI4NS4wNF9kZXNrdG9wLnBuZw==)
3. Выберите нужный доступ → «Дальше». ![](https://imgproxy.cdn-tinkoff.ru/compressed95:help_question_card_body_image_desktop_col_9/aHR0cHM6Ly9vcGlzLWNkbi50aW5rb2Zmam91cm5hbC5ydS9wbHV0by1iYWNrZW5kLWltYWdlcy8zMDY0NGRlNC4wNV9kZXNrdG9wLnBuZw==)
4. Выберите уровень доступа:
«С правом подписи» — соответствует уровню «Подписант». Для оформления понадобятся паспортные данные пользователя. Мы подготовим документы, чтобы выпустить электронную подпись.
«Без права подписи» — соответствует уровню «Менеджер». Для оформления не понадобятся паспортные данные пользователя. Он сможет смотреть информацию по счетам, создавать платежи и отправлять их на подпись.
[Подробнее про уровни доступа](/business/help/account/currency-ruble/personal-account/dashboard/?card=q7)
5. Заполните контактную информацию пользователя → «Дальше».
6. Введите код подтверждения из СМС → «Подтвердить». Готово! Вы добавили пользователя в Тинькофф Бизнес.
С компьютера — в личном кабинете на сайте tinkoff.ru
1. Нажмите на название компании или ИП → «Настройки профиля и доступы». ![](https://imgproxy.cdn-tinkoff.ru/compressed95:help_question_card_body_image_desktop_col_9/aHR0cHM6Ly9vcGlzLWNkbi50aW5rb2Zmam91cm5hbC5ydS9wbHV0by1iYWNrZW5kLWltYWdlcy9lODQxMDc4Zi5kZXNrdG9wXzQucG5n)
2. [Выберите вкладку «Компания» → «Все пользователи» → «Добавить»](https://business.tinkoff.ru/account/create-user) → «РФ» → «Дальше». ![](https://imgproxy.cdn-tinkoff.ru/compressed95:help_question_card_body_image_desktop_col_9/aHR0cHM6Ly9vcGlzLWNkbi50aW5rb2Zmam91cm5hbC5ydS9wbHV0by1iYWNrZW5kLWltYWdlcy84MTc0OTFkZi4wNl9kZXNrdG9wLnBuZw==)
3. Выберите нужный доступ → «Дальше». ![](https://imgproxy.cdn-tinkoff.ru/compressed95:help_question_card_body_image_desktop_col_9/aHR0cHM6Ly9vcGlzLWNkbi50aW5rb2Zmam91cm5hbC5ydS9wbHV0by1iYWNrZW5kLWltYWdlcy9iNDQ2ZmVlYy4wN19kZXNrdG9wLnBuZw==)
4. Выберите уровень доступа:
«С правом подписи» — соответствует уровню «Подписант». Для оформления понадобятся паспортные данные пользователя. Мы подготовим документы, чтобы выпустить электронную подпись.
«Без права подписи» — соответствует уровню «Менеджер». Для оформления не понадобятся паспортные данные пользователя. Он сможет смотреть информацию по счетам, создавать платежи и отправлять их на подпись.
[Подробнее про уровни доступа](/business/help/account/currency-ruble/personal-account/dashboard/?card=q7)
5. Заполните контактную информацию пользователя → «Дальше».
6. Введите код подтверждения из СМС → «Подтвердить». Готово! Вы добавили пользователя в Тинькофф Бизнес.
Доступ в аккаунт у пользователя появится сразу — ему придет СМС с паролем для входа.
Если вы добавили сотрудника с уровнем доступа «Подписант», он получит еще одно СМС, чтобы назначить встречу с представителем Тинькофф. Он привезет документы на подпись — от сотрудника потребуется только паспорт. После встречи с представителем право подписи появится в течение рабочего дня.
